# Contributing to Vaulthalla

Vaulthalla is Linux-native system software. It touches a C++ daemon, FUSE, PostgreSQL, TPM/swtpm, AppArmor, systemd, Debian packaging, Nginx, Certbot, and a packaged Next.js web runtime.

Contributions are welcome, but patches need to be scoped, validated, and honest about what they affect.

Start here:

- Contributor guide: `docs/contributors/README.md`
- AI-assisted contribution policy: `docs/contributors/ai-assisted-contributions.md`
- Validation guide: `docs/contributors/validation-guide.md`
- Roadmap: https://github.com/orgs/vaulthalla/projects/4
- Issues: https://github.com/vaulthalla/vaulthalla/issues
